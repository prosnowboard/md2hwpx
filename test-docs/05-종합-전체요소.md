## 5. 코드 블록

### Bash 스크립트

```bash
#!/bin/bash
# Oh-My-ClaudeCode 설치 스크립트

echo "Installing Oh-My-ClaudeCode..."
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode
/plugin install oh-my-claudecode
echo "Done! Run /oh-my-claudecode:omc-setup to configure."
```

### Python 코드

```python
class AgentRouter:
    """작업 복잡도에 따른 에이전트 라우팅"""

    TIER_MAP = {
        "simple": "haiku",
        "standard": "sonnet",
        "complex": "opus"
    }

    def route(self, task: str) -> str:
        complexity = self.analyze(task)
        return self.TIER_MAP.get(complexity, "sonnet")
```

### 인라인 코드

설정 파일은 `~/.claude/.omc-config.json`에 위치하며, `defaultExecutionMode` 값을 수정할 수 있습니다.
