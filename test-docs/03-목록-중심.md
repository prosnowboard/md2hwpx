# Oh-My-ClaudeCode 기능 목록

## 핵심 스킬 목록

### 자동 활성화 스킬

- **autopilot** — 완전 자율 실행
  - "autopilot", "build me", "I want a"로 트리거
  - 계획 → 실행 → 검증 → 완료까지 자동
- **ralph** — 완료까지 지속 실행
  - "don't stop", "must complete"로 트리거
  - 실패 시 자동 재시도
- **ultrawork** — 최대 병렬 실행
  - "fast", "parallel", "ulw"로 트리거
  - 독립적 작업 동시 처리
- **frontend-ui-ux** — UI/UX 디자인 감각
  - UI 작업 감지 시 자동 활성화 (사일런트)
- **git-master** — Git 전문가 모드
  - git/commit 관련 작업 시 자동 활성화 (사일런트)

### 수동 호출 스킬

1. **plan** — 전략적 계획 인터뷰
2. **ralplan** — 반복적 계획 합의 (Planner + Architect + Critic)
3. **analyze** — 심층 분석 및 조사
4. **deepsearch** — 코드베이스 심층 검색
5. **research** — 병렬 과학자 에이전트 오케스트레이션
6. **ultraqa** — QA 사이클링 (테스트/수정/반복)
7. **learner** — 세션에서 재사용 가능한 스킬 추출
8. **note** — 메모리 시스템에 저장

## MCP 서버 지원 목록

외부 도구와의 통합:

- **Context7**: 최신 라이브러리 문서 제공
- **Exa**: 향상된 검색 기능
- **GitHub API**: GitHub 통합
  - 이슈 관리
  - PR 생성 및 리뷰
  - 코드 검색
- **Filesystem**: 파일시스템 접근
  - 파일 읽기/쓰기
  - 디렉토리 탐색
  - 파일 검색

## HUD 상태 표시줄 기능

실시간으로 오케스트레이션 상태를 시각화:

1. 현재 활성 에이전트 표시
2. 작업 진행 상태 모니터링
3. 병렬 실행 현황 추적
4. 토큰 사용량 표시
5. 에러 및 경고 알림
